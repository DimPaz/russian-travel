# **Проект: Путешествие по России**

## Описание

[**Путешествие по России**](https://dimpaz.github.io/russian-travel/index.html) - это проект который вдохновит вас на путешествия. Наша страна красивая и удивительная, с богатой историей и традициями, изучать которые можно всю жизнь.

---

## Функционал

Интерфейс должен одинаково красиво и функционально работать на всех современных типах устройств:

- настольный компьютер
- ноутбук
- планшет
- смартфон

## Технологии:

- HTML5 (Flexbox, Grid Layout)
- CSS3 (Media Queries)
- Фйловая структура БЭМ Nested
- Верстка по макету Figma

#### Инструкция по использованию проекта

Для самостоятельной работы вам потребуется:

1. Программы VS Code, git-bash, YandexBrowser (или любой другой браузер).
2. [Скачать](https://github.com/DimPaz/russian-travel.git) репозиторий.
3. Ознакомиться с макетом [Figma](https://www.figma.com/file/5S2WSbEFL6awjVWJ0NWL8Q/Sprint-3_-Russia-_-desktop-mobile?node-id=28503%3A0).
4. Доставать картинки предстоит из Фигмы. Не забудьте [оптимизировать картинки](https://tinypng.com/), чтобы ваш сайт загружался быстрее.
